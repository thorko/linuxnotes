## remote desktop
**anydesk** => remote desktop app (works on linux)

## database
**TiDB** => scalable mysql database system

## network
**shodan** => network search tool  
**wicd** => wireless configuration tool  
**gns3** => network bgp simulator  
**softether** => vpn server and client  
**tinc** => meshed vpn client

**wireguard** => simple setup of vpn

## monitoring
**pnda** => monitor network devices with prediction  
**libreNMS** => network monitoring  
**nethogs** => which process using most bandwidth (traffic)  
**glances** => keep an eye on your linux system
**goaccess** => real time log analyzer (with nice graphs)
**sensu-go** => monitoring in volatile environments

**bmon** => show live statistics about network traffic

## productivity
**qownnotes** => app to create quickly simple notes  
**joplin** => another notes taking app for nextcloud
**kdesvn** => svn gui  
**filelight** => disk usage utility, integrated in dolphin, right below corner  
**recoll** => search and indexing tool for local files.  
**Vivaldi** => good customizable browser
**clickup** => project Management tool. Assign multiple assingees

## container
**kitematic** => docker GUI  
**portainer** => nice looking gui, manages local containers
**ctop** => watch resources of container in realtime

**velero** => backup your kubernetes cluster configuration

**cerebro** => elasticsearch gui to monitor indices and nodes in the cluster

## communication
**franz** => multi messaging app (supports Telegram, Threema, Mattermost, Slack...)

## misc
**atom** => ide, also used for markdown  
**visual studio code** => ide highly customizable and feature rich

**riva flv encoder** => convert movie files to other formats - Windows  
**avant** => nice tool bar looking like mac tiger bar - Linux  
**freetuxtv** => watch tv stations online - Linux  
**tlp** => save battery on linux  
**bleachbit** => like ccleaner for linux  
**odoo** => all in one office solution (crm)  
**sandstorm** => working suite
**sendgrid** => send email by provider

**masterpdfeditor4** => pdf editor
